# [Longest Collatz sequence](http://projecteuler.net/problem=14)
Scott Wiedemann

12/31/2013

## Compile It
ant


## Run It
java -jar ./target/lcs.jar N

Finds the starting value, less than N, in the longest Collatz sequence.  N is 1000000.
