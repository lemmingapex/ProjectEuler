[Largest product in a series](http://projecteuler.net/problem=8)
====================
Scott Wiedemann

07/13/2013

Compile It
----------
ant


Run It
------
./target/lps.jar N


Run-time
--------
N*m, where N is the number of consecutive digits in the product and m is the length of the digit string.  N is 5 and m is 1000 in this case.
